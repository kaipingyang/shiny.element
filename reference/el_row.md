# Element UI Layout Row

Element UI Layout Row

## Usage

``` r
el_row(
  ...,
  gutter = NULL,
  type = NULL,
  justify = NULL,
  align = NULL,
  class = NULL,
  style = NULL
)
```

## Arguments

- ...:

  Child columns (el_col) or other Element UI components

- gutter:

  Spacing between columns

- type:

  Layout type, e.g. "flex"

- justify:

  Flex horizontal alignment

- align:

  Flex vertical alignment

- class:

  CSS class

- style:

  CSS style
