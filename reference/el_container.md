# Element UI Container

Element UI Container

## Usage

``` r
el_container(
  ...,
  id = NULL,
  style = NULL,
  class = NULL,
  session = getDefaultReactiveDomain()
)
```

## Arguments

- ...:

  Child components

- id:

  Unique container id (auto-generated if NULL)

- style:

  CSS style

- class:

  CSS class

- session:

  Shiny session for module support
