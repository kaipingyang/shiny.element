# Element UI Layout Column

Element UI Layout Column

## Usage

``` r
el_col(
  ...,
  span = NULL,
  offset = NULL,
  push = NULL,
  pull = NULL,
  xs = NULL,
  sm = NULL,
  md = NULL,
  lg = NULL,
  xl = NULL,
  class = NULL,
  style = NULL
)
```

## Arguments

- ...:

  Content or other Element UI components

- span:

  Column span (1-24)

- offset:

  Offset columns

- push:

  Push columns

- pull:

  Pull columns

- xs:

  Responsive xs

- sm:

  Responsive sm

- md:

  Responsive md

- lg:

  Responsive lg

- xl:

  Responsive xl

- class:

  CSS class

- style:

  CSS style
