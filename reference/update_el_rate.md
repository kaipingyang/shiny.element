# Update Element UI Rate

Server-side update for
[`el_rate()`](https://kaipingyang.github.io/shiny.element/reference/el_rate.md).

## Usage

``` r
update_el_rate(session, id, value = NULL, disabled = NULL)
```

## Arguments

- session:

  Shiny session object.

- id:

  Rate ID (un-namespaced).

- value:

  New rating value.

- disabled:

  New disabled state.
