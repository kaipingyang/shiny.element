# Update Element UI Cascader

Update Element UI Cascader

## Usage

``` r
update_el_cascader(
  session,
  id,
  options = NULL,
  value = NULL,
  placeholder = NULL,
  clearable = NULL,
  filterable = NULL,
  disabled = NULL
)
```

## Arguments

- session:

  Shiny session object

- id:

  Cascader ID

- options:

  New cascader options

- value:

  New selected value

- placeholder:

  New placeholder text

- clearable:

  Whether clearable

- filterable:

  Whether filterable

- disabled:

  Whether disabled
